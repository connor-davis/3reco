# Introduction

**3rEco** is a real-time recycling marketplace platform that connects recycling collectors with businesses. Businesses source recyclable materials from collectors via direct purchases or the marketplace negotiation flow, while collectors track their revenue and sales on their personal dashboard.

## What is 3rEco?

The name stands for **3R** (Reduce, Reuse, Recycle) + **Eco**. The platform's goal is to formalise and digitise the secondary materials market, making it easier for businesses to source recyclables from collectors and for collectors to find reliable buyers.

## Key Features

- **Role-based access control** — four user types (Admin, Staff, Business, Collector) each with tailored dashboards and capabilities
- **Real-time marketplace** — transaction requests and negotiation messages update live via Convex subscriptions
- **Offer & negotiation flow** — businesses send purchase requests to sellers, negotiate weight and price, and complete trades
- **Automatic PDF invoices** — generated server-side and stored in Convex file storage upon transaction completion
- **Carbon savings tracking** — every material has a `carbonFactor`; transactions accumulate a carbon savings stat visible on dashboards
- **Data exports** — role-scoped CSV exports for transactions, stock, and materials
- **Notifications** — real-time, dismissible notification feed for all platform events

## Project Layout

```
3reco/           ← active React 19 + Convex app (this codebase)
app/             ← legacy SolidJS frontend (maintenance only)
api/             ← legacy Express.js + MongoDB REST API (maintenance only)
```

The `/3reco` package is the actively developed full-stack application where Convex replaces the legacy Express API entirely.
